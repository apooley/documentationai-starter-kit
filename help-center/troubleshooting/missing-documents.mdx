---
title: Documents Not Appearing
description: Fix missing documentation issues.
---

## Symptoms

- Created docs do not show on dashboard
- Newly added docs missing after refresh

## Causes

- Network error or sync delay
- Not saved properly

## Solutions

<Steps>
  <Step title="Refresh page">
    Try reloading the dashboard to update the list.
  </Step>
  <Step title="Check Save Status">
    Ensure you clicked 'Save' after editing.
  </Step>
  <Step title="Clear browser cache">
    Sometimes caching can hide new content.
  </Step>
</Steps>
